# Geneta
Application to Simulate and visualize the Mendelian inheritance.


## How to use
- Start a new Simulation
- Select the Traits you would like to have Simulated
- Select the Dominace Type of each Trait

After that, you'll be able to select 2 Flowers. On the Left side of the Screen, you can pair those selected Flower and generate a new Generation.
Also on the left menu, a new Simulation can be started.

The ! on the Flowers shows the Genotype.

Have fun!
